# Enlightenment — Сайт практик и ретритов

Сайт-визитка с золотистой божественной эстетикой для практик в Священной Долине.

## Версии

- **Версия 1** (`1/`) — с изображениями (services.png, who-we-are.png, schedule.png)
- **Версия 2** (`2/`) — полностью интерактивная (без изображений)
- **Версия 3** (`3/`) — интерактив + качественные фото (живой сайт)
- **Версия 4** (`4/`) — просветлённая: стабильная, без дёрганий
- **Для reg.ru** (`hosting/`) — простая: HTML + CSS, без внешних зависимостей

## Запуск

```bash
npx serve .
```

Откройте http://localhost:3000 — выбор версии. Или сразу `1/index.html` / `2/index.html`.

## Структура

```
visitka/
├── index.html      # Выбор версии
├── 1/              # Версия с изображениями
├── 2/              # Интерактивная версия
└── hosting/        # Для reg.ru (HTML + CSS, img/)
```
