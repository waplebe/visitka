# Enlightenment — Демо-сайт практик и ретритов

Сайт-визитка с золотистой божественной эстетикой для практик в Священной Долине.

## Как обновить контент

### Недельное расписание
Замените файл `assets/schedule.png` — загрузите новое изображение с расписанием с тем же именем. Сайт автоматически покажет обновлённую картинку.

### QR-код для связи
Добавьте файл `assets/qr-contact.png` с вашим QR-кодом (WhatsApp, Telegram и т.д.). Рекомендуемый размер: 200×200 px.

### Контактная информация
Отредактируйте блок «Свяжитесь с нами» в `index.html` — замените плейсхолдеры на реальные Telegram, WhatsApp, Email.

### Текст «Кто мы»
Измените текст в секции `#who-we-are` в `index.html` под ваше описание.

## Запуск

Откройте `index.html` в браузере или используйте локальный сервер:

```bash
npx serve .
```

## Структура

- `index.html` — основная страница
- `styles.css` — стили (золотистая палитра)
- `assets/` — изображения (services.png, who-we-are.png, schedule.png, qr-contact.png)
